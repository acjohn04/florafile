# Playbook Page

**Route**: `/playbook` (mapped to `src/app/(main)/playbook/page.tsx`)
**Rendering**: Client Component (`"use client"`)

---

## Overview

The playbook page generates a personalized, AI-powered care schedule for all plants in the user's household and exports it as a downloadable `.ics` calendar file. The schedule covers 3 months and is compatible with Google Calendar, Apple Calendar, and Outlook.

---

## Functionality

### Schedule Generation

Clicking "Download Smart Schedule" triggers the following flow:

1. `window.location.href` is set to `/api/playbook/export` (triggers a browser download)
2. The API route:
   - Fetches all plants for the authenticated user's household
   - Calls `generateCareSchedule()` from `src/lib/gemini.ts` with plant data and today's date
   - Gemini generates up to 50 care tasks across 3 months, considering each plant's watering frequency and care level
   - Tasks are mapped to `CalendarTask` objects with 9:00 AM start times
   - `generateICS()` converts tasks to the iCalendar (.ics) format
3. The response is returned with `Content-Type: text/calendar` and `Content-Disposition: attachment` headers
4. The browser downloads `florafile-smart-schedule.ics`

### Loading State

A spinner and "Generating..." text replace the download button during processing. The loading state auto-resets after 3 seconds (since native downloads can't easily signal completion to JavaScript).

### Empty State

If the user has no plants, an empty ICS file is returned (valid calendar with no events).

---

## AI Schedule Generation

The Gemini prompt instructs the AI to:
- Act as a "master gardener"
- Space out tasks to avoid overwhelming the owner on any single day
- Consider each plant's `water` frequency and `careLevel`
- Generate tasks like "Water Monstera" with descriptions like "Check top 2 inches of soil first"
- Return dates in `YYYY-MM-DD` format
- Limit to max 50 events total

---

## ICS Calendar Format

Generated by `src/lib/calendar.ts`:
- Each task becomes a `VEVENT` with a 1-hour duration (9:00–10:00 AM)
- UIDs are random to avoid conflicts on re-import
- Standard `VCALENDAR` wrapper with `PRODID:-//FloraFile//EN`

---

## API Endpoint

### `GET /api/playbook/export`

- **Auth**: Requires authenticated session via `requireHousehold()`
- **Output**: `.ics` file download
- **AI model**: `gemini-2.5-flash`
- **Task limit**: 50 events maximum
- **Schedule span**: 3 months from today
